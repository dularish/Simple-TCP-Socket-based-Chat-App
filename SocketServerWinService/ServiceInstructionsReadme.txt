To install the service :

1. Open Developer command prompt for Visual studio 20XX in administrator mode
2. Navigate to the built directory (bin\Debug or bin\Release)
3. Install the service using the command "installutil SocketServerWinService.exe"

To uninstall the service :

1. Open Developer command prompt for Visual studio 20XX in administrator mode
2. Navigate to the built directory (bin\Debug or bin\Release)
3. Uninstall the service using the command "installutil /u SocketServerWinService.exe"
